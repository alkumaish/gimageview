10. GImageView 更新履歴

2002-11-04: Version 0.2.7
    o ファイル操作時に，ファイル操作をキャンセル/失敗してもキャッシュファイルと
      コメントファイルが移動/コピー/リンクされてしまう問題を修正
    o 異なるファイルシステムへファイルを移動する時に，元のファイルを削除するよう
      に戻した
	 o リネームモードのファイル名エントリでキーアクセラレータが無効にならない
      問題を修正(Gtk+2使用時)
    o リネームモードで文字コードが正しく変換されない問題を修正(Gtk+2使用時)
    o サムネイルビューでサムネイルのフォーカス移動が正しく動作しない問題を修正
      (Gtk+2使用時)

2002-11-02: Version 0.2.6
    o Gtk+-2.0サポートの改善(主にGtkTreeViewへの対応，その他バグフィックス多数)
    o 新しいソートメニューを追加
    o サムネイルビューでディレクトリを表示した状態で再帰ローディングすると無限
      ループになる問題を修正
    o サムネイルウィンドウに「キャッシュをクリア」メニューを追加
    o ZAlbumウィジェットにおいてマルチバイト文字列を考慮していなかった問題を修正．
    o コメントデータのファイル名の文字コード変換が誤っていた問題を修正．
    o コード整理
    o その他細かい修正

2002-09-22: Version 0.2.5
    o サムネイルの作成速度を改善
    o デフォルトのサムネイル表示モードを「サムネイル2」に変更
    o サムネイル読み込み済みのディレクトリをハイライトするようにした
    o 「詳細」モードに画像サイズを表示できるようにした
    o ディレクトリビューのマウスボタン設定を追加
    o イメージウィンドウの背景色を設定できるようにした
    o コンパイルエラーを修正(Thanks! 下森さん & えづらさん)
    o その他細かい修正

2002-09-02: Version 0.2.4
    o 「全画面」の挙動を変更
    o ドキュメントのインストールディレクトリを変更
    o configure/コンパイルエラーの修正
    o Nautilus-2.0のサムネイルに対応
    o XVサムネイルの他プログラムとの互換性を向上
    o Gtk+-2.0サポートの改善
    o その他バグフィックス多数

2002-07-01: Version 0.2.3
    o アニメーションGIFのサポートを追加 (gdk-pixbuf使用時)
    o MNG/JNGサポートを追加 (要libmng)
    o 試験的な圧縮ファイルのサポート (gzip, bz2) この機能を使用したい場合は、
      設定->一般->フィルタに "gz"及び"bz2"を追加して下さい
    o フランス語カタログファイルを追加 (Thanks! Guillaume Libersatさん)
    o サムネイルロード時に，表示領域内のサムネイルを優先してロードするようにした
    o その他バグフィックス

2002-05-31: Version 0.2.2
    o SVGサポートを追加(librsvgが必要)
    o 試験的な動画ファイルサポートを追加(xine-libが必要)
    o 試験的なSusieプラグインサポートを追加(Broken)
    o Gtk+-2.0対応を改善
    o デスクトップファイルを追加
    o その他細かい改善

2002-05-08: Version 0.2.1
    o 「サムネイル2」モードにおいて、マウスドラッグによる範囲選択時の反応を向上、

    o 「リスト2」モードを追加(「サムネイル2」と同じZAlbumウィジェットを使用)
    o ${datadir}/gimageview/gtkrc を読み込むようにした
    o ローディング中はタブラベルのスタイルを変更するようにした
    o イメージウィンドウを開いた直後に閉じると落ちる問題を修正
    o サムネイルサイズを変更してから類似画像検索結果ウィンドウを閉じると落ちる問題を修正
    o nsさんからの指摘で、specファイルのtypoを修正

    o 類似画像検索機能を追加
    o 画像保存/サムネイル関連機能をプラグイン化

    o その他細かい修正多数
