7. デフォルト キーバインド

7.1. 共通

  • B : タブ表示 ON/OFF

  • F : 全画面

  • M : メニューバー表示 ON/OFF

  • N : ツールバー表示 ON/OFF

  • V : ステータスバー表示 ON/OFF

  • Ctrl + C : GImageView 全終了

  • Ctrl + F : ファイルオープンダイアログ

  • Ctrl + I : 新規イメージウィンドウ

  • Ctrl + O : 設定

  • Ctrl + Q : ウィンドウを閉じる

  • Ctrl + T : 新規タブ

  • Ctrl + W : 新規ウィンドウ

  • Shift + S : スクロールバー表示 ON/OFF

  • Ctrl + Shift + F10 : ポップアップメニュー

7.2. サムネイルウィンドウ

  • H : 前のページ(タブ)

  • J : 最後のページ(タブ)

  • K : 最初のページ(タブ)

  • L : 次のページ(タブ)

  • Q : タブを閉じる

  • R : リスト更新

  • Ctrl + 0 : レイアウト 0

  • Ctrl + 1 : レイアウト 1

  • Ctrl + 2 : レイアウト 2

  • Ctrl + 3 : レイアウト 3

  • Ctrl + 4 : レイアウト 4

  • Ctrl + A : 全て選択

  • Ctrl + D : タブを切り離す

  • Ctrl + N : 全て選択解除

  • Ctrl + Q : ウィンドウを閉じる

  • Ctrl + R : キャッシュ再作成

  • Ctrl + S : スライドショー

  • Ctrl + T : 新規タブ

  • Shift + C : ディレクトリへコピー

  • Shift + B : タブを後方へ移動

  • Shift + D : ディレクトリビュー表示 ON/OFF

  • Shift + E : ファイル名変更

  • Shift + F : タブを前方へ移動

  • Shift + L : ディレクトリへリンク

  • Shift + M : ディレクトリへ移動

  • Shift + N : ディレクトリビューツールバー表示 ON/OFF

  • Shift + P : プレビュー表示 ON/OFF

  • Shift + R : キャッシュ再読込

  • Shift + Z : プレビューツールバー表示 ON/OFF

7.3. サムネイルウィンドウ / サムネイルビュー

  • カーソルキー : カーソル移動

  • Space : ファイルを選択/選択解除

  • Enter : そのファイルのみを選択して画像を開く

  • Shift + Enter : そのファイルを選択して画像を開く

  • Delete : 選択ファイルを削除

ただし、ファイル選択の挙動はモードによって少し異なる場合があります。

  • 2 : 25%

  • 3 : 50%

  • 7 : 150%

  • A : 5%縮小

  • E : 90度回転

  • R : -90度回転

  • S : 5%拡大

  • W : ウィンドウの大きさに合わせる

  • Ctrl + B : メモリバッファ

  • Shift + A : アスペクト比を保持

  • Shift + T : サムネイル作成

  • Page Up : 前の画像

  • Page Down, Space : 次の画像

  • Home : 最初の画像

  • End : 最後の画像

  • カーソルキー : 画像移動

ズーム及び回転はナビゲーションウィンドウでも有効です。

7.6. マウス

7.6.1. サムネイルビュー

  • ボタン 1 (左クリック) : 開く (Window を自動選択)

  • Ctrl + ボタン 1 (Ctrl + 左クリック) : none

  • Shift + ボタン 1 (Shift + 左クリック) : クリックした画像まで一気に選択。選択解除は出来ません。全て選択解除(Ctrl+N)してください。

  • Alt + ボタン 1 (Alt + 左クリック) : none

  • ボタン 2 (右クリック) : 開く (自動 & 強制)

  • Ctrl + ボタン 2 (Ctrl + 右クリック) : none

  • Shift + ボタン 2 (Shift + 右クリック) : none

  • Alt + ボタン 2 (Alt + 右クリック) : none

  • ボタン 3 (中クリック) : ポップアップメニュー

  • Ctrl + ボタン 3 (Ctrl + 中クリック) : none

  • Shift + ボタン 3 (Shift + 中クリック) : none

  • Alt + ボタン 3 (Alt + 中クリック) : none

  • ボタン 1 ダブルクリック (左ダブルクリック) : 開く (自動 & 強制)

  • Ctrl + ボタン 1 ダブルクリック (Ctrl + 左ダブルクリック) : none

  • Shift + ボタン 1 ダブルクリック (Shift + 左ダブルクリック) : none

  • Alt + ボタン 1 ダブルクリック (Alt + 左ダブルクリック) : none

  • ボタン 4 (ホイールアップ) : none

  • Ctrl + ボタン 4 (Ctrl + ホイールアップ) : none

  • Shift + ボタン 4 (Shift + ホイールアップ) : none

  • Alt + ボタン 4 (Alt + ホイールアップ) : none

  • ボタン 5 (ホイールダウン) : none

  • Ctrl + ボタン 5 (Ctrl + ホイールダウン) : none

  • Shift + ボタン 5 (Shift + ホイールダウン) : none

  • Alt + ボタン 5 (Alt + ホイールダウン) : none

7.6.3. ディレクトリビュー

  • ボタン 1 (左クリック) : ディレクトリ選択

  • ボタン 1 ([+]の部分) (左クリック) : ディレクトリツリー展開

  • ボタン 2 (右クリック) : ポップアップメニュー

  • ボタン 3 (中クリック) : ディレクトリ展開。ディレクトリ下に画像があった場合、サムネイル表示も併せて実行

  • ボタン 1 ダブルクリック (左ダブルクリック) : そのディレクトリ直下をサムネイル表示。子ディレクトリを持っている場合、ディレクトリツリー展開も併せて実行
